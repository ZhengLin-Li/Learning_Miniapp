# Yet another way to implement binary search that makes your life eaiser
* WITH ans VARIABLE!!!*

```java
class Solution {
    public int[] searchRange(int[] nums, int target) {
        return new int[]{searchRange(nums, target, true), searchRange(nums, target, false)};
    }

    public int searchRange(int[] nums, int target, boolean lower) {
        if (nums == null || nums.length == 0) {
            return -1;
        }
        int ans = -1;
        // [left, right]
        int left = 0, right = nums.length - 1;
        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (nums[mid] == target) {
                if (lower) {
                    ans = mid;
                    right = mid - 1;
                } else {
                    ans = mid;
                    left = mid + 1;
                }
            } else if (nums[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        if (ans == -1) return -1;
        return ans;
    }
}
```

## Yet another way to understand backtrack and dfs
```java
// DFS 算法，关注点在节点
void traverse(TreeNode root) {
    if (root == null) return;
    printf("进入节点 %s", root);
    for (TreeNode child : root.children) {
        traverse(child);
    }
    printf("离开节点 %s", root);
}

// 回溯算法，关注点在树枝
void backtrack(TreeNode root) {
    if (root == null) return;
    for (TreeNode child : root.children) {
        // 做选择
        printf("从 %s 到 %s", root, child);
        backtrack(child);
        // 撤销选择
        printf("从 %s 到 %s", child, root);
    }
}
```

```java
        for (int key : numToFre.keySet()) {
```
