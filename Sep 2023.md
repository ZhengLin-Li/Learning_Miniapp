## Yet another way to use heap

original way:
```java
        for (int key : numToFre.keySet()) {
            if (pq.size() < k) {
                pq.offer(key);
            } else {
                int value = numToFre.get(key);
                if (value > numToFre.get(pq.peek())) {
                    pq.poll();
                    pq.offer(key);
                }
            }
        }
```


optimized way:
```java
        for (Integer key : map.keySet()) {
            pq.offer(key);
            if (pq.size() > k) {
                pq.poll();
            }
        }
```
